# Data Notes

This folder contains processed school-level modelling artefacts derived from publicly available Department for Education datasets.

No individual pupil-level data is included.

The key files are:

- `school_panel_final.parquet` — final engineered school-level panel
- `model_input.parquet` — modelling input used for final training/evaluation
- `urn_mapping.pkl` — institutional mapping artefact used for entity resolution
